# 🏰 魔塔世界观设定

## 故事背景

### 起源传说

在远古时代，众神创造了一座通天神塔，塔中蕴藏着无穷的力量。然而，贪婪的人类为了争夺塔中的力量，引发了旷日持久的战争。最终，众神降下诅咒，将神塔封印，化作一座充满考验的魔塔。

传说中，只有真正勇敢、智慧且心怀正义的人，才能登上塔顶，获得神的认可。

### 魔塔结构

魔塔共 **50 层**，分为五个区域：

| 区域 | 楼层 | 名称 | 特点 |
|------|------|------|------|
| I | 1-10 | 试炼之塔 | 新手区域，基础怪物和道具 |
| II | 11-20 | 迷雾之塔 | 迷宫复杂，需要策略 |
| III | 21-30 | 暗影之塔 | 强大敌人，隐藏要素 |
| IV | 31-40 | 炼狱之塔 | 极高难度，特殊机制 |
| V | 41-50 | 神之塔 | 最终考验，传说真相 |

## 主角设定

### 基本信息

| 项目 | 内容 |
|------|------|
| **姓名** | 艾伦（玩家可自定义） |
| **年龄** | 20 岁 |
| **身份** | 王国骑士团见习骑士 |
| **动机** | 寻找失踪的父亲（前代勇者） |
| **特点** | 勇敢、正直、善于思考 |

### 背景故事

艾伦的父亲是王国传说中的勇者，曾在 20 年前挑战魔塔，但从此失踪。多年来，艾伦一直相信父亲还活着，被困在魔塔某处。

成年后，艾伦继承父亲的意志，踏上挑战魔塔的旅程。他不仅要解开魔塔的秘密，更要找到父亲的下落。

---

## 重要 NPC

### 1. 神秘老人（第 1 层）

- **身份**：魔塔守护者/引导者
- **作用**：提供初始指导，出售基础道具
- **台词**：*「年轻人，你准备好面对塔中的考验了吗？」*

### 2. 被困的公主（第 10 层）

- **身份**：邻国公主，探险时被困
- **作用**：提供关键道具，后续剧情线索
- **台词**：*「谢谢你救了我！但这只是塔的开始...」*

### 3. 神秘商人（随机楼层）

- **身份**：神秘商人，只在特定条件出现
- **作用**：出售稀有道具
- **台词**：*「有缘人，我这里有你需要东西...」*

### 4. 黑暗骑士（第 25 层）

- **身份**：前代挑战者，被诅咒腐蚀
- **作用**：Boss 战，剧情转折
- **台词**：*「你也想追求力量吗？看看我的下场吧...」*

### 5. 艾伦的父亲（第 50 层）

- **身份**：前代勇者，真相守护者
- **作用**：最终剧情揭示
- **台词**：*「孩子，你终于来了。现在，你该知道真相了...」*

## 魔塔规则

### 基本法则

1. **固定数值**：塔中一切战斗结果都可预先计算
2. **资源有限**：钥匙、血瓶等道具需要合理分配
3. **不可回头**：下楼容易，但某些门只能单向通过
4. **智慧至上**：蛮力无法通关，需要策略规划

### 特殊机制

| 机制 | 说明 |
|------|------|
| **幸运楼层** | 某些楼层有额外奖励 |
| **隐藏房间** | 特定条件触发隐藏要素 |
| **时间限制** | 部分区域有步数限制 |
| **连击系统** | 连续无伤战斗有奖励 |

## 多人模式设定（创新要素）

### 协作模式

2-4 名玩家组队进入魔塔：
- **共享视野**：可以看到队友位置
- **资源分配**：钥匙和道具可交易
- **合力战斗**：部分 Boss 需要配合
- **复活机制**：队友可互相救援

### 对抗模式

多名玩家竞速爬塔：
- **独立进度**：各自挑战，互不干扰
- **排行榜**：实时显示排名
- **干扰道具**：可有限度影响对手
- **速通记录**：个人最佳时间追踪

## 主题与寓意

### 核心主题

- **勇气**：面对未知挑战的勇气
- **智慧**：策略胜过蛮力
- **坚持**：永不放弃的精神
- **亲情**：父子羁绊
