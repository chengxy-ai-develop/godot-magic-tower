# 🏗️ 魔塔框架开发计划

**分支:** `feature/framework-dev`  
**创建时间:** 2026-03-14 17:21  
**基于分支:** `main` (魔塔 1.0 完整版)

---

## 📋 开发目标

将魔塔游戏框架化，创建可复用的游戏引擎，支持：
- 可视化关卡编辑器
- MOD 系统 (玩家自制内容)
- 插件系统 (功能扩展)
- 快速创建新魔塔游戏

## 🔧 技术要点

### 1. 核心框架
- 单例模式 (GameManager, EventBus)
- 信号驱动 (Godot Signal)
- 数据驱动 (JSON 配置)

### 2. 编辑器
- Godot EditorPlugin
- 自定义 Inspector
- 实时预览

### 3. MOD 系统
- ZIP 解压
- JSON 解析
- 资源热加载

### 4. 插件系统
- 钩子机制
- 事件拦截
- API 沙箱
