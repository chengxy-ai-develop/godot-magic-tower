## 🎮 游戏简介

《魔塔》是一款经典固定数值策略 RPG 游戏的原创重制版。玩家需要逐层爬塔，通过策略性的资源管理和战斗规划，最终到达塔顶。

### 核心特色

- 🎯 **经典玩法** - 固定数值策略，可精确计算每一步
- 🗺️ **原创剧情** - 独立世界观，非复刻经典魔塔
- 👥 **多人模式** - 支持协作爬塔和竞速对抗
- 🎨 **现代高清** - 精美高清美术风格
- 📱 **跨平台** - PC + 移动端全支持
- 🆓 **免费开源** - MIT 协议，欢迎贡献

## 🚀 快速开始

### 环境要求

- **Godot Engine**: 4.2 或更高版本
- **Git**: 用于版本控制

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/chengxy-ai-develop/godot-magic-tower.git

# 2. 进入项目目录
cd godot-magic-tower

# 3. 使用 Godot 打开项目
# 在 Godot 中选择"导入"，然后选择 project.godot 文件
```

### 运行游戏

1. 在 Godot 编辑器中打开项目
2. 按 `F5` 运行
3. 或导出为可执行文件

## 🎯 游戏操作

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `E` / `空格` | 交互/确认 |
| `ESC` | 菜单/取消 |
