## 🔧 技术实现

### 架构设计
```
┌─────────────────────────────────────┐
│           Main.tscn                 │
│  (主场景 - 游戏入口)                 │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│          GameScene.gd               │
│  (游戏场景 - 系统整合)               │
└──┬─────────────┬─────────────┬──────┘
   │             │             │
┌──▼──────┐  ┌──▼──────┐  ┌──▼──────┐
│ Player  │  │GameManager│ │ UILayer │
└─────────┘  └─────────┘  └────┬────┘
                               │
                    ┌──────────▼──────────┐
                    │    UIManager        │
                    │ (背包/状态/商店/对话)│
                    └─────────────────────┘
```

### 核心系统
1. **GameManager** - 游戏状态、流程控制、战斗解析
2. **Player** - 移动、属性、物品、装备、等级
3. **BattleSystem** - 自动战斗、伤害计算
4. **SaveSystem** - 多槽位存档、JSON 序列化
5. **UIManager** - 所有 UI 界面管理
6. **MapLoader** - 从 JSON 生成地图

### 数据流
```
JSON 数据 → MapLoader → 场景实例化
    ↓
GameManager → 游戏状态管理
    ↓
Player + Enemy → 战斗系统
    ↓
UIManager → 界面更新
    ↓
SaveSystem → 存档保存
```
